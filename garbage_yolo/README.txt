К вашему вниманию будет представлено две модели(обучено через YOLO)(обученные на 100 эпох)
Которые должна сортировать мусор на изображение по следующим категориям:

1 - alu - алюминий( в нашем случае алюминиевые банки);
2 - carton - картон;
3 - glass - стекло( в нашем случае стеклянные бутылки);
4 - mask - медицинские маски;
5 - paper_cup - бумажные стаканчики;
6 - pla_cup - пластиковые стаканчики;
7 - pla_flasche - пластиковые бутылки;
8 - tin_cans - жестяные банки.

Также в файле config.yaml расписаны классы для обучения.
Код для обучения модели помещён для образца - Yolo11.py
У меня не добавлены здесь файлы(датасет) для обучения модели, но сам код обучения добавлен для наглядности
Папка garbage_dataset_yolo пустая, в неё не добавлено датасета.

2 обученные модели: 
Первая модель находится - train/weights/best.pt 
Вторая модель находится - train2/weights/best.pt 

Код для проверки моделей на различных изображениях в папке testik - Test2.py
В коде приведены некоторые пояснения для удобства использования моделей.

Также важно установить библиотеки:
Это можно сделать благодаря файлу requirements.txt
Чтобы установить все нужные библиотеки вам нужно ввести просто pip install -r requirements.txt 

После выполнения кода Test2.py будут выходить изображения и там выводится результат: что модель увидела на изображение.
Я выполнял это в PyCharm.
